У цьому розділі подано додаткові приклади подій для повнішого розуміння їхнього використання.

Приклади

Подія виходу гравця

phpCopy code
use pocketmine\\event\\player\\PlayerQuitEvent;
use pocketmine\\event\\Listener;


class MyEventListener implements Listener {


    public function onPlayerQuit(PlayerQuitEvent $event) {
        $player = $event->getPlayer();
        $event->setQuitMessage("До побачення, {$player->getName()}!");
    }
}

Цей приклад обробляє подію виходу гравця і встановлює повідомлення про вихід із певним текстом.

Подія зміни часу в грі

phpCopy code
use pocketmine\\event\\level\\LevelTimeEvent;
use pocketmine\\event\\Listener;


class MyTimeListener implements Listener {


    public function onLevelTimeChange(LevelTimeEvent $event) {
        $level = $event->getLevel();
        $time = $event->getTime();


        if ($time % 24000 === 0) {
            $level->setTime(0); // Обнулення часу кожні 24 000 тиків (новий день)
        }
    }
}

Цей приклад дає змогу вам обробляти подію зміни часу в грі і, наприклад, обнуляти час кожного нового дня.

Подія смерті сутності

phpCopy code
use pocketmine\\event\\entity\\EntityDeathEvent;
use pocketmine\\event\\Listener;


class MyDeathListener implements Listener {


    public function onEntityDeath(EntityDeathEvent $event) {
        $entity = $event->getEntity();
        $cause = $entity->getLastDamageCause();


        if ($cause instanceof EntityDamageByEntityEvent) {
            $killer = $cause->getDamager();
            $entity->getLevel()->dropItem($entity, Item::get(Item::DIAMOND, 0, 1));
            $killer->sendMessage("Ви вбили {$entity->getName()} і отримали алмаз!");
        }
    }
}

Цей приклад обробляє подію смерті сутності та випадання предмета при вбивстві сутності гравцем.

Ці приклади надають лише малу частину можливостей обробки подій у PocketMine-MP. Розглянуті події можуть бути базою для створення більш складних і цікавих механік у вашому сервері.

Створення простого обробника події

У цьому розділі ми створимо простий обробник події для більш детального розуміння процесу.

  1. Створення плагіна та реєстрація обробника:
    phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\\plugin\\PluginBase;
use pocketmine\\event\\Listener;

class MyPlugin extends PluginBase implements Listener {

    public function onEnable() {
        $this->getServer()->getPluginManager()->registerEvents($this, $this);
    }
}
  1. Створення класу-обробника:
    phpCopy code
// MyEventListener.php

namespace MyPlugin;

use pocketmine\\event\\player\\PlayerJoinEvent;
use pocketmine\\event\\Listener;

class MyEventListener implements Listener {

    public function onPlayerJoin(PlayerJoinEvent $event) {
        $player = $event->getPlayer();
        $player->sendMessage("Ласкаво просимо, {$player->getName()}!");
    }
}
  1. Підключення обробника в основному класі:
    phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\\plugin\\PluginBase;
use pocketmine\\event\\Listener;

class MyPlugin extends PluginBase implements Listener {

    public function onEnable() {
        $this->getServer()->getPluginManager()->registerEvents(new MyEventListener(), $this);
    }
}

Цей приклад створює простий обробник події PlayerJoinEvent, який надсилає вітальне повідомлення під час приєднання гравця до сервера. Ви можете використовувати аналогічний підхід для обробки інших подій і створення складніших функцій у вашому плагіні.

Реальні приклади використання подій у плагінах

У цьому розділі розглянемо реальні приклади використання подій у плагінах для PocketMine-MP.

Оповіщення про вхід і вихід гравця

phpCopy code
// JoinQuitNotifier.php


namespace MyPlugin;


use pocketmine\\plugin\\PluginBase;
use pocketmine\\event\\Listener;
use pocketmine\\event\\player\\PlayerJoinEvent;
use pocketmine\\event\\player\\PlayerQuitEvent;


class JoinQuitNotifier extends PluginBase implements Listener {


    public function onEnable() {
        $this->getServer()->getPluginManager()->registerEvents($this, $this);
    }


    public function onPlayerJoin(PlayerJoinEvent $event) {
        $player = $event->getPlayer();
        $player->sendMessage("Ласкаво просимо, {$player->getName()}!");
    }


    public function onPlayerQuit(PlayerQuitEvent $event) {
        $player = $event->getPlayer();
        $this->getServer()->broadcastMessage("Гравець {$player->getName()} вийшов із гри.");
    }
}

Цей приклад демонструє використання подій PlayerJoinEvent і PlayerQuitEvent для оповіщення гравця про його вхід і надсилання широкомовного повідомлення при виході гравця.

Захист регіону від руйнування блоків*.

phpCopy code
// RegionProtection.php


namespace MyPlugin;


use pocketmine\\plugin\\PluginBase;
use pocketmine\\event\\Listener;
use pocketmine\\event\\block\\BlockBreakEvent;


class RegionProtection extends PluginBase implements Listener {


    private $protectedRegions = ["spawn", "safe_zone"];


    public function onEnable() {
        $this->getServer()->getPluginManager()->registerEvents($this, $this);
    }


    public function onBlockBreak(BlockBreakEvent $event) {
        $block = $event->getBlock();
        $regionName = $this->getRegionName($block->getX(), $block->getZ());


        if (in_array($regionName, $this->protectedRegions)) {
            $event->setCancelled(true);
            $event->getPlayer()->sendMessage("Цей регіон захищений від руйнування блоків.");
        }
    }


    private function getRegionName($x, $z) {
        // Ваш код для визначення імені регіону за координатами
    }
}

У цьому прикладі використовується подія BlockBreakEvent для запобігання руйнуванню блоків у певних захищених регіонах.

Ці приклади надають уявлення про те, як події використовуються для створення різних функцій у плагінах для PocketMine-MP. Ваша креативність і потреби сервера визначать, які саме події ви будете використовувати і як їх обробляти.
