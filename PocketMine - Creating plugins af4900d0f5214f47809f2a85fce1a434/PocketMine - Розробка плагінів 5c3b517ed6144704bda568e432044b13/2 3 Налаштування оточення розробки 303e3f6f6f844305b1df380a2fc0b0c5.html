Для успішного створення плагінів для сервера PocketMine-MP вам необхідно налаштувати ваше оточення розробки. Це охоплює встановлення необхідних інструментів, таких як PHP і Composer, а також вибір і налаштування інтегрованого середовища розробки (IDE). Давайте розберемо кожен крок.

Встановлення PHP і Composer

Встановлення PHP:

  1. Вибір Версії PHP:
    • Визначте необхідну версію PHP для вашого проекту. Рекомендується використовувати PHP 7.2 і вище.
  1. Завантаження та Встановлення PHP:
    • Відвідайте офіційний сайт PHP і завантажте необхідну версію PHP для вашої операційної системи.
  1. Встановлення на Windows:
    • Запустіть інсталяційний файл PHP і дотримуйтесь інструкцій майстра встановлення. Переконайтеся, що додали PHP у змінну середовища PATH.
  1. Встановлення на Linux:
    • Скористайтеся менеджером пакетів вашого дистрибутива для встановлення PHP, наприклад:
      sudo apt-get install php7.4

Встановлення Composer:

  1. Завантаження та встановлення Composer:
    • Відвідайте офіційний сайт Composer і дотримуйтесь інструкцій для завантаження та встановлення Composer.
  1. Перевірка Встановлення:
    • У консолі виконайте команду composer --version, щоб переконатися, що Composer успішно встановлено.

Налаштування IDE для комфортної розробки

Вибір і Встановлення IDE:

  1. Вибір IDE:
    • Виберіть інтегроване середовище розробки (IDE) для роботи з кодом плагінів. Рекомендованими варіантами є PhpStorm, Visual Studio Code, або Eclipse.
  1. Встановлення IDE:
    • Завантажте та встановіть обрану IDE з офіційного сайту.

Налаштування Проекту в IDE:

  1. Відкриття Проекту:
    • Відкрийте IDE і виберіть опцію "Open Project" або "Відкрити Проект". Вкажіть шлях до кореневої папки вашого сервера PocketMine-MP.
  1. Налаштування SDK:
    • Вкажіть версію PHP, яка встановлена на вашому сервері, в налаштуваннях проекту.

Додавання Плагіна в Проект:

  1. Створення Папки для Плагіна:
    • У корені сервера створіть папку для вашого плагіна, наприклад, "MyPlugin".
  1. Створення Файлу-Плагіна:
    • У створеній папці створіть файл плагіна, наприклад, "MyPlugin.php".
  1. Відкриття Проекту в IDE:
    • Відкрийте створений файл в IDE.
  1. Розробка Плагіна:
    • Почніть розробляти ваш плагін, використовуючи можливості обраної IDE. Реалізуйте необхідну функціональність подій, команд та інших аспектів.

Підключення IDE до Сервера (Опціонально):

  1. Встановлення Плагіна XDebug:
    • Якщо плануєте налагодження, встановіть плагін XDebug у вашу IDE.
  1. Налаштування Підключення:
    • Налаштуйте з'єднання IDE з вашим сервером, використовуючи XDebug. Вкажіть IP-адресу сервера і порт для підключення до відладчика.

Тепер ваше середовище розробки готове до створення та налагодження плагінів для сервера PocketMine-MP. У наступних розділах ми розглянемо створення плагінів та їхні основні концепції.
