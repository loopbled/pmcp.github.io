Створення власних команд у PocketMine-MP дає вам змогу надавати гравцям унікальний функціонал і керування вашим плагіном. Давайте розглянемо процес створення та реєстрації власних команд.

Кроки зі створення простої команди

Створення простої команди в PocketMine-MP включає кілька кроків. Давайте розглянемо основні кроки, необхідні для створення та реєстрації простої команди.

Крок 1: Створення класу команди

Створіть клас для вашої команди, наприклад, SimpleCommand.php:

phpCopy code
// SimpleCommand.php


namespace MyPlugin\\command;


use pocketmine\\command\\Command;
use pocketmine\\command\\CommandSender;


class SimpleCommand extends Command {


    public function __construct() {
        parent::__construct("simple", "Проста команда", "/simple");
    }


    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        $sender->sendMessage("Привіт, це проста команда!");
        return true;
    }
}

Крок 2: Реєстрація команди в основному класі плагіна

Тепер зареєструйте вашу просту команду в основному класі вашого плагіна (MyPlugin.php):

phpCopy code
// MyPlugin.php


namespace MyPlugin;


use pocketmine\\plugin\\PluginBase;
use pocketmine\\command\\CommandSender;
use pocketmine\\command\\Command;
use MyPlugin\\command\\SimpleCommand;


class MyPlugin extends PluginBase {


    public function onEnable() {
        $this->getLogger()->info("Плагін успішно увімкнено!");


        // Реєстрація простої команди
        $this->getServer()->getCommandMap()->register("simple", new SimpleCommand());
    }
}

Крок 3: Використання простої команди в грі

Тепер гравці можуть використовувати вашу просту команду. Наприклад:

bashCopy code
# Введення в чаті гри
/simple

Коли гравець введе команду /simple, йому прийде відповідь "Привіт, це проста команда!".

Ці кроки надають базовий каркас для створення простої команди в PocketMine-MP. Ви можете доповнити логіку виконання команди в методі execute класу команди відповідно до ваших потреб.

Додавання параметрів і обробка введення користувача

При створенні команд у PocketMine-MP ви можете додавати параметри, щоб команда стала більш гнучкою і функціональною. Давайте розглянемо, як додати параметри та обробити введення користувача.

Крок 1: Розширення класу команди з параметрами

Розширте клас вашої команди (SimpleCommand.php), щоб додати параметри:

phpCopy code
// SimpleCommand.php


namespace MyPlugin\\command;


use pocketmine\\command\\Command;
use pocketmine\\command\\CommandSender;


class SimpleCommand extends Command {


    public function __construct() {
        parent::__construct("simple", "Проста команда з параметрами", "/simple <параметр1> <параметр2>");
    }


    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        // Перевірка наявності необхідної кількості параметрів
        if (count($args) < 2) {
            $sender->sendMessage("Використання: /simple <параметр1> <параметр2>");
            return false;
        }


        $param1 = $args[0];
        $param2 = $args[1];


        // Логіка виконання команди з використанням параметрів
        $sender->sendMessage("Ви ввели параметр1: $param1 і параметр2: $param2");


        return true;
    }
}

Крок 2: Реєстрація оновленої команди в основному класі плагіна

Зареєструйте оновлену команду в основному класі плагіна (MyPlugin.php):

phpCopy code
// MyPlugin.php


namespace MyPlugin;


use pocketmine\\plugin\\PluginBase;
use pocketmine\\command\\CommandSender;
use pocketmine\\command\\Command;
use MyPlugin\\command\\SimpleCommand;


class MyPlugin extends PluginBase {


    public function onEnable() {
        $this->getLogger()->info("Плагін успішно увімкнено!");


        // Реєстрація оновленої команди
        $this->getServer()->getCommandMap()->register("simple", new SimpleCommand());
    }
}

Крок 3: Використання команди з параметрами в грі

Тепер гравці можуть використовувати вашу команду з параметрами. Наприклад:

bashCopy code
# Введення в чаті гри
/simple значення1 значення2

У цьому прикладі команда очікує на два параметри (значення1 і значення2), і виводить повідомлення з їхніми значеннями.

Додавання параметрів дає змогу створювати більш гнучкі та потужні команди у ваших плагінах для PocketMine-MP.
