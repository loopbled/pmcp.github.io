Робота з командами надає можливість додавати користувацькі команди у ваш плагін для PocketMine-MP. У цьому модулі ми розглянемо основи створення та обробки команд.

Проста команда

Створення простої команди:

phpCopy code
// MyCommandPlugin.php


namespace MyPlugin;


use pocketmine\\plugin\\PluginBase;
use pocketmine\\command\\Command;
use pocketmine\\command\\CommandSender;


class MyCommandPlugin extends PluginBase {


    public function onEnable() {
        $this->getLogger()->info("Плагін успішно увімкнено!");
    }


    public function onCommand(CommandSender $sender, Command $command, string $label, array $args): bool {
        if ($command->getName() === "hello") {
            $sender->sendMessage("Привіт, {$sender->getName()}!");
            return true;
        }
        return false;
    }
}

Реєстрація команди в основному класі плагіна:

phpCopy code
// MyCommandPlugin.php


namespace MyPlugin;


use pocketmine\\plugin\\PluginBase;
use pocketmine\\command\\Command;
use pocketmine\\command\\CommandSender;


class MyCommandPlugin extends PluginBase {


    public function onEnable() {
        $this->getLogger()->info("Плагін успішно увімкнено!");
        $this->getServer()->getCommandMap()->register("hello", new MyCommand());
    }
}

Використання команди в грі:

  • Запустіть сервер зі встановленим плагіном.
  • У грі введіть команду /hello.

Це всього лише простий приклад створення команди. Ваш плагін може мати більш складні команди з додатковими параметрами і функціональністю.

Передача аргументів команді:

phpCopy code
// MyCommandPlugin.php


namespace MyPlugin;


use pocketmine\\plugin\\PluginBase;
use pocketmine\\command\\Command;
use pocketmine\\command\\CommandSender;


class MyCommandPlugin extends PluginBase {


    public function onEnable() {
        $this->getLogger()->info("Плагін успішно увімкнено!");
        $this->getServer()->getCommandMap()->register("greet", new GreetCommand());
    }
}



phpКопіювати код
// GreetCommand.php


namespace MyPlugin\\command;


use pocketmine\\command\\Command;
use pocketmine\\command\\CommandSender;
use pocketmine\\Player;


class GreetCommand extends Command {


    public function __construct() {
        parent::__construct("greet", "Привітання гравця", "/greet <гравець>");
        $this->setPermission("myplugin.command.greet");
    }


    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        if (!$this->testPermission($sender)) {
            return false;
        }


        if (empty($args)) {
            $sender->sendMessage("Використовуйте: /greet <гравець>");
            return false;
        }


        $player = $sender->getServer()->getPlayer($args[0]);
        if ($player instanceof Player) {
            $sender->sendMessage("Привіт, {$player->getName()}!");
        } else {
            $sender->sendMessage("Гравця {$args[0]} не знайдено.");
        }


        return true;
    }
}

Цей приклад демонструє команду /greet, яка вітає вказаного гравця.

Робота з командами відкриває безліч можливостей для керування плагіном і спілкування з гравцями на сервері. Ви можете адаптувати наведені приклади для створення складніших команд і функцій у своєму плагіні.

Введення в систему команд у Minecraft

У Minecraft команди являють собою потужний інструмент для взаємодії з ігровим світом і керування сервером. Команди виконуються через чат або консоль сервера і надають різноманітні можливості, включно з телепортацією, видачею предметів, керуванням часом доби та багато іншого.

Основні елементи команд:

  1. Команди:
    • Команди починаються з косої риски (/).
    • Приклад: /tp Player1 Player2 - телепортує гравця Player1 до гравця Player2.
  1. Аргументи:
    • Команди можуть містити аргументи, що являють собою дані або параметри.
    • Приклад: /give Player1 diamond 64 - видає гравцеві Player1 64 алмази.
  1. Параметри:
    • Деякі команди мають додаткові параметри для налаштування їхньої роботи.
    • Приклад: /time set day - встановлює час доби на день.

Приклад використання команд у Minecraft:

  1. Телепортація:
    • /tp Player1 Player2 - телепортує гравця Player1 до гравця Player2.
  1. Видача предметів:
    • /give Player1 diamond 64 - видає гравцеві Player1 64 алмази.
  1. Керування часом:
    • /time set day - встановлює час доби на день.
  1. Зміна ігрового режиму:
    • /gamemode creative - встановлює ігровий режим Creative.
  1. Зміна погоди:
    • /weather clear - встановлює ясну погоду.

Робота з командами в PocketMine-MP:

У PocketMine-MP ви можете створювати свої власні команди за допомогою плагінів. Це дає змогу розширити функціональність сервера і надати гравцям унікальні можливості. У наступних розділах ми розглянемо створення та керування командами в PocketMine-MP.

Реєстрація базових команд

Реєстрація базових команд у PocketMine-MP надає зручний спосіб взаємодії з вашим плагіном. У цьому розділі ми розглянемо процес створення та реєстрації простих команд.

Крок 1: Створення класу команди

Для кожної команди необхідно створити відповідний клас. Наприклад, розглянемо команду /hello:

phpCopy code
// HelloCommand.php

namespace MyPlugin\\command;

use pocketmine\\command\\Command;
use pocketmine\\command\\CommandSender;

class HelloCommand extends Command {

    public function __construct() {
        parent::__construct("hello", "Привітання від плагіна", "/hello");
    }

    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        $sender->sendMessage("Привіт, {$sender->getName()}!");
        return true;
    }
}

Крок 2: Реєстрація команди в основному класі плагіна

Тепер ми повинні зареєструвати нашу команду в основному класі плагіна:

phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\\plugin\\PluginBase;
use pocketmine\\command\\CommandSender;
use pocketmine\\command\\Command;
use MyPlugin\\command\\HelloCommand;

class MyPlugin extends PluginBase {

    public function onEnable() {
        $this->getLogger()->info("Плагін успішно увімкнено!");

        // Реєстрація команди
        $this->getServer()->getCommandMap()->register("hello", new HelloCommand());
    }
}

Крок 3: Використання команди в грі

Тепер гравці можуть використовувати команду /hello для отримання привітання від плагіна.

bashCopy code
# Введення в чаті гри
/hello

Це лише простий приклад. Ви можете створювати більш складні команди з додатковими параметрами та функціональністю відповідно до потреб вашого плагіна.
