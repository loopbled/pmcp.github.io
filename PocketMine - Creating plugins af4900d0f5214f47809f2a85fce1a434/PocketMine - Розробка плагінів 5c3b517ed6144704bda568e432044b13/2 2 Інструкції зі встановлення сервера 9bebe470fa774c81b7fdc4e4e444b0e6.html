Локальна установка для тестування

Підготовка до встановлення

Перед початком встановлення сервера PocketMine-MP для локального тестування, переконайтеся, що у вас встановлені PHP і Git. Ви можете завантажити PHP з офіційного сайту PHP, а Git - з офіційного сайту Git.

Клонування репозиторію PocketMine-MP

  1. відкрийте командний рядок (термінал) на вашому комп'ютері.
  1. Перейдіть у каталог, де ви хочете встановити сервер.
  1. Виконайте таку команду:
    bashCopy code
git clone --recursive <https://github.com/pmmp/PocketMine-MP.git>

    Ця команда клонує репозиторій PocketMine-MP на ваш комп'ютер.

Встановлення залежностей

  1. Перейдіть у каталог PocketMine-MP:
    bashCopy code
cd PocketMine-MP
  1. Запустіть встановлення залежностей:
    bashКопіювати код
./start.sh

    Ця дія встановить необхідні залежності для коректної роботи сервера.

Запуск сервера

  1. Поверніться в кореневий каталог PocketMine-MP.
  1. Запустіть сервер:
    bashCopy code
./start.sh

    Коли ви бачите повідомлення, що підтверджує успішний запуск сервера, це означає, що сервер готовий до використання.

Тепер ваш локальний сервер PocketMine-MP встановлений і готовий до тестування. Підключіться до сервера з Minecraft: Bedrock Edition, використовуючи IP-адресу "localhost" і порт, зазначений у файлі server.properties.

Розміщення сервера на віддаленому хостингу

Вибір і підготовка Хостингу

  1. Вибір Хостингу:
    • Виберіть хостинг-провайдера, який підтримує PHP і надає можливість розміщення Minecraft-серверів. Популярні варіанти включають Akliz, PebbleHost та інші.
  1. Реєстрація на Хостингу:
    • Зареєструйтеся на обраному хостингу, дотримуючись їхніх інструкцій.
  1. Отримання Доступу до Хостингу:
    • Отримайте доступ до вашого облікового запису на хостингу і знайдіть розділ управління серверами.

Встановлення та налаштування PocketMine-MP

  1. Отримання Доступу до Консолі:
    • На панелі керування вашого хостингу знайдіть вкладку "Console" або "Консоль". Отримайте доступ до консолі вашого сервера.
  1. Клонування Репозиторію PocketMine-MP:
    • Введіть таку команду в консоль, щоб схилити репозиторій PocketMine-MP:
      bashCopy code
git clone --recursive <https://github.com/pmmp/PocketMine-MP.git>
  1. Встановлення Залежностей:
    • Перейдіть у каталог PocketMine-MP:
      bashКопіювати код
cd PocketMine-MP
    • Запустіть встановлення залежностей:
      bashCopy code
./start.sh

Запуск Сервера на Хостингу

  1. Налаштування server.properties:
    • Відредагуйте файл server.properties у кореневому каталозі сервера, встановивши основні параметри.
  1. Запуск Сервера:
    • Поверніться в кореневий каталог PocketMine-MP і запустіть сервер:
      bashCopy code
./start.sh
    • Переконайтеся, що сервер успішно запущений і готовий до прийняття підключень.
  1. Підключення до Сервера:
    • Поверніться на панель керування хостингом і знайдіть IP-адресу та порт вашого сервера.
    • У Minecraft: Bedrock Edition підключіться до сервера, використовуючи знайдену IP-адресу та порт.

Тепер ваш сервер PocketMine-MP розміщений на віддаленому хостингу і готовий до використання. Підключайтеся і починайте налаштовувати свій ігровий світ.
