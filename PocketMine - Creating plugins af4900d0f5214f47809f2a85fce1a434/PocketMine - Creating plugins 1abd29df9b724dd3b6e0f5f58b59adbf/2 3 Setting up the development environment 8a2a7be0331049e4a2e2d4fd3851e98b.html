To successfully create plugins for the PocketMine-MP server, you need to customize your development environment. This includes installing the necessary tools, such as PHP and Composer, as well as selecting and customizing your integrated development environment (IDE). Let's break down each step.

Install PHP and Composer

Install PHP:

  1. Select PHP Version:.
    • Determine the required PHP version for your project. PHP 7.2 or higher is recommended.
  1. Download and Install PHP:
    • Visit the official PHP website and download the required version of PHP for your operating system.
  1. Installation on Windows:
    • Run the PHP installation file and follow the instructions in the installation wizard. Make sure to add PHP to the PATH environment variable.
  1. Installing on Linux:
    • Use your distribution's package manager to install PHP, for example:
      sudo apt-get install php7.4

Install Composer:

  1. Download and Install Composer:
    • Visit the official Composer website and follow the instructions to download and install Composer.
  1. Check Installation:
    • In the console, run the composer --version command to verify that Composer is successfully installed.

Customize the IDE for comfortable development

Select and Install IDE:

  1. Selecting an IDE:
    • Select an integrated development environment (IDE) to work with the plugin code. Recommended options are PhpStorm, Visual Studio Code, or Eclipse.
  1. Installing the IDE:
    • Download and install the selected IDE from the official website.

Customize the Project in the IDE:

  1. Open Project:
    • Open the IDE and select the "Open Project" or "Open Project" option. Specify the path to the root folder of your PocketMine-MP server.
  1. Customize SDK:
    • Specify the PHP version that is installed on your server in the project settings.

Adding a Plugin to the Project:

  1. Creating a Folder for the Plugin:
    • In the root of the server, create a folder for your plugin, for example, "MyPlugin".
  1. Creating a Plugin File:
    • In the created folder, create a plugin file, for example, "MyPlugin.php".
  1. Opening the Project in IDE:
    • Open the created file in the IDE.
  1. Developing the Plugin:
    • Start developing your plugin using the features of the selected IDE. Realize the necessary functionality of events, commands, and other aspects.

Connect IDE to Server (Optional):

  1. Installing the XDebug Plugin:
    • If you plan to debug, install the XDebug plugin in your IDE.
  1. Configuring the Connection:
    • Set up the IDE connection to your server using XDebug. Specify the server IP address and port to connect to the debugger.

Your development environment is now ready to create and debug plugins for the PocketMine-MP server. In the following sections we will consider the creation of plug-ins and their basic concepts.
