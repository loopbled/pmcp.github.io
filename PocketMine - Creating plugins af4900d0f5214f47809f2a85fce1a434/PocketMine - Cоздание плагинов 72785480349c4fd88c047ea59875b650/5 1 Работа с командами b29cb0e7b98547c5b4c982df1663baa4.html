Работа с командами предоставляет возможность добавлять пользовательские команды в ваш плагин для PocketMine-MP. В этом модуле мы рассмотрим основы создания и обработки команд.

Простая команда

Создание простой команды:

phpCopy code
// MyCommandPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\command\Command;
use pocketmine\command\CommandSender;

class MyCommandPlugin extends PluginBase {

    public function onEnable() {
        $this->getLogger()->info("Плагин успешно включен!");
    }

    public function onCommand(CommandSender $sender, Command $command, string $label, array $args): bool {
        if ($command->getName() === "hello") {
            $sender->sendMessage("Привет, {$sender->getName()}!");
            return true;
        }
        return false;
    }
}

Регистрация команды в основном классе плагина:

phpCopy code
// MyCommandPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\command\Command;
use pocketmine\command\CommandSender;

class MyCommandPlugin extends PluginBase {

    public function onEnable() {
        $this->getLogger()->info("Плагин успешно включен!");
        $this->getServer()->getCommandMap()->register("hello", new MyCommand());
    }
}

Использование команды в игре:

  • Запустите сервер с установленным плагином.
  • В игре введите команду /hello.

Это всего лишь простой пример создания команды. Ваш плагин может иметь более сложные команды с дополнительными параметрами и функциональностью.

Передача аргументов команде:

phpCopy code
// MyCommandPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\command\Command;
use pocketmine\command\CommandSender;

class MyCommandPlugin extends PluginBase {

    public function onEnable() {
        $this->getLogger()->info("Плагин успешно включен!");
        $this->getServer()->getCommandMap()->register("greet", new GreetCommand());
    }
}


phpCopy code
// GreetCommand.php

namespace MyPlugin\command;

use pocketmine\command\Command;
use pocketmine\command\CommandSender;
use pocketmine\Player;

class GreetCommand extends Command {

    public function __construct() {
        parent::__construct("greet", "Приветствие игрока", "/greet <игрок>");
        $this->setPermission("myplugin.command.greet");
    }

    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        if (!$this->testPermission($sender)) {
            return false;
        }

        if (empty($args)) {
            $sender->sendMessage("Используйте: /greet <игрок>");
            return false;
        }

        $player = $sender->getServer()->getPlayer($args[0]);
        if ($player instanceof Player) {
            $sender->sendMessage("Привет, {$player->getName()}!");
        } else {
            $sender->sendMessage("Игрок {$args[0]} не найден.");
        }

        return true;
    }
}

Этот пример демонстрирует команду /greet, которая приветствует указанного игрока.

Работа с командами открывает множество возможностей для управления плагином и общения с игроками на сервере. Вы можете адаптировать приведенные примеры для создания более сложных команд и функций в своем плагине.

Введение в систему команд в Minecraft

В Minecraft команды представляют собой мощный инструмент для взаимодействия с игровым миром и управления сервером. Команды выполняются через чат или консоль сервера и предоставляют разнообразные возможности, включая телепортацию, выдачу предметов, управление временем суток и многое другое.

Основные элементы команд:

  1. Команды:
    • Команды начинаются с косой черты (/).
    • Пример: /tp Player1 Player2 - телепортирует игрока Player1 к игроку Player2.
  1. Аргументы:
    • Команды могут содержать аргументы, представляющие собой данные или параметры.
    • Пример: /give Player1 diamond 64 - выдает игроку Player1 64 алмаза.
  1. Параметры:
    • Некоторые команды имеют дополнительные параметры для настройки их работы.
    • Пример: /time set day - устанавливает время суток на день.

Пример использования команд в Minecraft:

  1. Телепортация:
    • /tp Player1 Player2 - телепортирует игрока Player1 к игроку Player2.
  1. Выдача предметов:
    • /give Player1 diamond 64 - выдает игроку Player1 64 алмаза.
  1. Управление временем:
    • /time set day - устанавливает время суток на день.
  1. Изменение игрового режима:
    • /gamemode creative - устанавливает игровой режим Creative.
  1. Изменение погоды:
    • /weather clear - устанавливает ясную погоду.

Работа с командами в PocketMine-MP:

В PocketMine-MP вы можете создавать свои собственные команды с помощью плагинов. Это позволяет расширить функциональность сервера и предоставить игрокам уникальные возможности. В следующих разделах мы рассмотрим создание и управление командами в PocketMine-MP.

Регистрация базовых команд

Регистрация базовых команд в PocketMine-MP предоставляет удобный способ взаимодействия с вашим плагином. В этом разделе мы рассмотрим процесс создания и регистрации простых команд.

Шаг 1: Создание класса команды

Для каждой команды необходимо создать соответствующий класс. Например, рассмотрим команду /hello:

phpCopy code
// HelloCommand.php

namespace MyPlugin\command;

use pocketmine\command\Command;
use pocketmine\command\CommandSender;

class HelloCommand extends Command {

    public function __construct() {
        parent::__construct("hello", "Приветствие от плагина", "/hello");
    }

    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        $sender->sendMessage("Привет, {$sender->getName()}!");
        return true;
    }
}

Шаг 2: Регистрация команды в основном классе плагина

Теперь мы должны зарегистрировать нашу команду в основном классе плагина:

phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\command\CommandSender;
use pocketmine\command\Command;
use MyPlugin\command\HelloCommand;

class MyPlugin extends PluginBase {

    public function onEnable() {
        $this->getLogger()->info("Плагин успешно включен!");

        // Регистрация команды
        $this->getServer()->getCommandMap()->register("hello", new HelloCommand());
    }
}

Шаг 3: Использование команды в игре

Теперь игроки могут использовать команду /hello для получения приветствия от плагина.

bashCopy code
# Ввод в чате игры
/hello

Это лишь простой пример. Вы можете создавать более сложные команды с дополнительными параметрами и функциональностью в соответствии с потребностями вашего плагина.
