В этом разделе представлены дополнительные примеры событий для более полного понимания их использования.

Примеры

Событие выхода игрока

phpCopy code
use pocketmine\event\player\PlayerQuitEvent;
use pocketmine\event\Listener;

class MyEventListener implements Listener {

    public function onPlayerQuit(PlayerQuitEvent $event) {
        $player = $event->getPlayer();
        $event->setQuitMessage("До свидания, {$player->getName()}!");
    }
}

Этот пример обрабатывает событие выхода игрока и устанавливает сообщение о выходе с определенным текстом.

Событие изменения времени в игре

phpCopy code
use pocketmine\event\level\LevelTimeEvent;
use pocketmine\event\Listener;

class MyTimeListener implements Listener {

    public function onLevelTimeChange(LevelTimeEvent $event) {
        $level = $event->getLevel();
        $time = $event->getTime();

        if ($time % 24000 === 0) {
            $level->setTime(0); // Обнуление времени каждые 24 000 тиков (новый день)
        }
    }
}

Этот пример позволяет вам обрабатывать событие изменения времени в игре и, например, обнулять время каждый новый день.

Событие смерти сущности

phpCopy code
use pocketmine\event\entity\EntityDeathEvent;
use pocketmine\event\Listener;

class MyDeathListener implements Listener {

    public function onEntityDeath(EntityDeathEvent $event) {
        $entity = $event->getEntity();
        $cause = $entity->getLastDamageCause();

        if ($cause instanceof EntityDamageByEntityEvent) {
            $killer = $cause->getDamager();
            $entity->getLevel()->dropItem($entity, Item::get(Item::DIAMOND, 0, 1));
            $killer->sendMessage("Вы убили {$entity->getName()} и получили алмаз!");
        }
    }
}

Этот пример обрабатывает событие смерти сущности и выпадение предмета при убийстве сущности игроком.

Эти примеры предоставляют лишь малую часть возможностей обработки событий в PocketMine-MP. Рассмотренные события могут быть базой для создания более сложных и интересных механик в вашем сервере.

Создание простого обработчика события

В этом разделе мы создадим простой обработчик события для более детального понимания процесса.

  1. Создание плагина и регистрация обработчика:
    phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\event\Listener;

class MyPlugin extends PluginBase implements Listener {

    public function onEnable() {
        $this->getServer()->getPluginManager()->registerEvents($this, $this);
    }
}
  1. Создание класса-обработчика:
    phpCopy code
// MyEventListener.php

namespace MyPlugin;

use pocketmine\event\player\PlayerJoinEvent;
use pocketmine\event\Listener;

class MyEventListener implements Listener {

    public function onPlayerJoin(PlayerJoinEvent $event) {
        $player = $event->getPlayer();
        $player->sendMessage("Добро пожаловать, {$player->getName()}!");
    }
}
  1. Подключение обработчика в основном классе:
    phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\event\Listener;

class MyPlugin extends PluginBase implements Listener {

    public function onEnable() {
        $this->getServer()->getPluginManager()->registerEvents(new MyEventListener(), $this);
    }
}

Этот пример создает простой обработчик события PlayerJoinEvent, который отправляет приветственное сообщение при присоединении игрока к серверу. Вы можете использовать аналогичный подход для обработки других событий и создания более сложных функций в вашем плагине.

Реальные примеры использования событий в плагинах

В этом разделе рассмотрим реальные примеры использования событий в плагинах для PocketMine-MP.

Оповещение о входе и выходе игрока

phpCopy code
// JoinQuitNotifier.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\event\Listener;
use pocketmine\event\player\PlayerJoinEvent;
use pocketmine\event\player\PlayerQuitEvent;

class JoinQuitNotifier extends PluginBase implements Listener {

    public function onEnable() {
        $this->getServer()->getPluginManager()->registerEvents($this, $this);
    }

    public function onPlayerJoin(PlayerJoinEvent $event) {
        $player = $event->getPlayer();
        $player->sendMessage("Добро пожаловать, {$player->getName()}!");
    }

    public function onPlayerQuit(PlayerQuitEvent $event) {
        $player = $event->getPlayer();
        $this->getServer()->broadcastMessage("Игрок {$player->getName()} вышел из игры.");
    }
}

Этот пример демонстрирует использование событий PlayerJoinEvent и PlayerQuitEvent для оповещения игрока о его входе и отправке широковещательного сообщения при выходе игрока.
