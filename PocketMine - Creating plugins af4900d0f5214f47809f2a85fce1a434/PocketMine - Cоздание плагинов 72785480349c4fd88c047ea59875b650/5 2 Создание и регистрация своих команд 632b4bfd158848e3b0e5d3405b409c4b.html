Создание собственных команд в PocketMine-MP позволяет вам предоставлять игрокам уникальный функционал и управление вашим плагином. Давайте рассмотрим процесс создания и регистрации собственных команд.

Шаги по созданию простой команды

Создание простой команды в PocketMine-MP включает несколько шагов. Давайте рассмотрим основные шаги, необходимые для создания и регистрации простой команды.

Шаг 1: Создание класса команды

Создайте класс для вашей команды, например, SimpleCommand.php:

phpCopy code
// SimpleCommand.php

namespace MyPlugin\command;

use pocketmine\command\Command;
use pocketmine\command\CommandSender;

class SimpleCommand extends Command {

    public function __construct() {
        parent::__construct("simple", "Простая команда", "/simple");
    }

    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        $sender->sendMessage("Привет, это простая команда!");
        return true;
    }
}

Шаг 2: Регистрация команды в основном классе плагина

Теперь зарегистрируйте вашу простую команду в основном классе вашего плагина (MyPlugin.php):

phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\command\CommandSender;
use pocketmine\command\Command;
use MyPlugin\command\SimpleCommand;

class MyPlugin extends PluginBase {

    public function onEnable() {
        $this->getLogger()->info("Плагин успешно включен!");

        // Регистрация простой команды
        $this->getServer()->getCommandMap()->register("simple", new SimpleCommand());
    }
}

Шаг 3: Использование простой команды в игре

Теперь игроки могут использовать вашу простую команду. Например:

bashCopy code
# Ввод в чате игры
/simple

Когда игрок введет команду /simple, ему придет ответ "Привет, это простая команда!".

Эти шаги предоставляют базовый каркас для создания простой команды в PocketMine-MP. Вы можете дополнить логику выполнения команды в методе execute класса команды в соответствии с вашими потребностями.

Добавление параметров и обработка ввода пользователя

При создании команд в PocketMine-MP вы можете добавлять параметры, чтобы команда стала более гибкой и функциональной. Давайте рассмотрим, как добавить параметры и обработать ввод пользователя.

Шаг 1: Расширение класса команды с параметрами

Расширьте класс вашей команды (SimpleCommand.php), чтобы добавить параметры:

phpCopy code
// SimpleCommand.php

namespace MyPlugin\command;

use pocketmine\command\Command;
use pocketmine\command\CommandSender;

class SimpleCommand extends Command {

    public function __construct() {
        parent::__construct("simple", "Простая команда с параметрами", "/simple <параметр1> <параметр2>");
    }

    public function execute(CommandSender $sender, string $commandLabel, array $args): bool {
        // Проверка наличия необходимого количества параметров
        if (count($args) < 2) {
            $sender->sendMessage("Использование: /simple <параметр1> <параметр2>");
            return false;
        }

        $param1 = $args[0];
        $param2 = $args[1];

        // Логика выполнения команды с использованием параметров
        $sender->sendMessage("Вы ввели параметр1: $param1 и параметр2: $param2");

        return true;
    }
}

Шаг 2: Регистрация обновленной команды в основном классе плагина

Зарегистрируйте обновленную команду в основном классе плагина (MyPlugin.php):

phpCopy code
// MyPlugin.php

namespace MyPlugin;

use pocketmine\plugin\PluginBase;
use pocketmine\command\CommandSender;
use pocketmine\command\Command;
use MyPlugin\command\SimpleCommand;

class MyPlugin extends PluginBase {

    public function onEnable() {
        $this->getLogger()->info("Плагин успешно включен!");

        // Регистрация обновленной команды
        $this->getServer()->getCommandMap()->register("simple", new SimpleCommand());
    }
}

Шаг 3: Использование команды с параметрами в игре

Теперь игроки могут использовать вашу команду с параметрами. Например:

bashCopy code
# Ввод в чате игры
/simple значение1 значение2

В этом примере команда ожидает два параметра (значение1 и значение2), и выводит сообщение с их значениями.

Добавление параметров позволяет создавать более гибкие и мощные команды в ваших плагинах для PocketMine-MP.
